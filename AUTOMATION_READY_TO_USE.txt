LANGKAH 1: SETUP (HANYA 1x)
─────────────────────────────────────────────────────────────────────────────
1. Install jika belum ada:
   - PHP: https://www.php.net/downloads
   - Node.js: https://nodejs.org/
   - Git: https://git-scm.com/
   - GitHub account: https://github.com

2. Setup Git credentials di Command Prompt:
   git config --global user.name "Nama Anda"
   git config --global user.email "email@anda.com"

3. Verify terinstall:
   php -v
   node -v
   git --version


LANGKAH 2: SETUP REPOSITORY (HANYA 1x)
─────────────────────────────────────────────────────────────────────────────
1. Pastikan folder project sudah di-init sebagai Git repo:
   git status  (check apakah sudah ada .git folder)

2. Jika belum ada, buat repo di GitHub:
   - https://github.com/new
   - Create repository
   - Clone ke komputer lokal

3. Done! Repository siap untuk automation!


LANGKAH 3: DEPLOY AUTOMATION (SETIAP KALI ADA PERUBAHAN)
─────────────────────────────────────────────────────────────────────────────
1. Edit file di VS Code

2. Buka Command Prompt

3. Navigate ke folder project:
   cd "c:\Users\usedr\Desktop\APLIKASI BUMI SULTAN TERBARU\presensiv2\presensigpsv2-main\gawe.bumisultan"

4. Jalankan automation script:
   deploy-auto-full.bat

5. Ketik pesan commit (atau Enter untuk auto message):
   Update: Deskripsi perubahan

6. Tekan Enter dan TUNGGU...
   Script akan otomatis:
   ├─ Deploy lokal (update deps, build frontend)
   ├─ Test semua dependency
   ├─ Git add semua files
   ├─ Git commit dengan pesan Anda
   ├─ Git push ke GitHub
   └─ Show summary

7. SELESAI! Kode sudah di GitHub ✅

8. Buka Termius:
   - Connect ke hosting
   - cd /home/bumisultan/public_html
   - git pull origin main && npm run build
   - php artisan cache:clear
   - Done! Live di website ✅
